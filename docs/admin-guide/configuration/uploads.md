
## File Uploads Location
Directory for storing temporary uploaded files, including but not limited to product images, 
shipment documents, data imports.

```shell
openboxes:
    uploads:
        location: "uploads"
```

### Customization

```shell
openboxes:
    uploads:
        location: "uploads"
```
