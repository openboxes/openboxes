## Mail Configuration
Configure email settings for sending emails from the application.
```yaml
grails:
    mail:
        host: smtp.example.com
        port: 587
        username: your-email@example.com
        password: your-email-password
        props:
            "mail.smtp.auth": "true"
            "mail.smtp.starttls.enable": "true"
```
