jQuery Watermark Plugin

v 3.1.3

This jQuery plugin will easily create watermark hints in input and textarea elements.

Features:

  • Watermark style is controlled by CSS classes
  • Each input element can have its own independent watermark text and class/style
  • Plugin automatically removes all watermarks prior to form submission
  • Both watermark text and CSS class name can be changed dynamically
  • Visual Studio Intellisense-compatible documentation included in source
  • Comes with jQuery selector extension ":data(...)" (documented in source)
  • Password input automatically switches back to obfuscated text when focused
  • Minified version is very compact — under 4,400 bytes

Demonstrations:

  • Demo 1 – Simple watermark
  • Demo 2 – Several watermarks created with one statement + different class name
  • Demo 3 – Watermark changed dynamically (countdown timer)
  • Demo 4 – Multi-line watermark in a TEXTAREA tag
  • Demo 5 – Watermark in a password input element
  • Demo 6 – Support for WebKit search element (type="search")
  • Demo 7 – .val() gets/sets underlying element value
